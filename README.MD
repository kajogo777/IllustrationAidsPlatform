## Getting Started

# installation
1. install mongoDB
2. clone repository
3. cd ./illustration_aids_library
4. cd ./server
5. npm install
6. cd ..
7. cd ./client
8. npm install

# start up the application
1. run mongod
2. cd ./illustration_aids_library/server
3. npm strat
4. cd ./illustration_aids_library/server
5. npm start
6. browse to "http://localhost:3000/"
